# Seed Classification Project (Machine Learning)

## Objective
Build a machine learning model to classify Sacha Inchi seeds using image data.

## Tools
- Python
- TensorFlow
- Keras
- OpenCV
- Pandas

## Process
- Data collection from embedded systems (Raspberry Pi / Arduino)
- Image preprocessing and cleaning
- Model training using CNN
- Evaluation and optimization

## Outcome
Developed a convolutional neural network capable of classifying seed images with reliable accuracy.

## Skills Demonstrated
- Data preprocessing
- Image classification
- Machine learning (CNN)
- Data pipeline handling
