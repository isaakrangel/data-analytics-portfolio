# Machine Learning Final Project

## Objective
Develop a machine learning regression model to predict taxi fare amounts using Python.

## Tools
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  

## Process
- Data cleaning and preprocessing  
- Feature engineering  
- Exploratory Data Analysis (EDA)  
- Feature selection  
- Model training and evaluation  
- Performance optimization  

## Models Used
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- Gradient Boosting Regressor  
- Cross-validation for performance evaluation  

## Results
- The model achieved good overall predictive performance  
- Higher prediction errors were observed for high fare values, with slight underestimation  
- Consistent performance was observed on training and unseen data, indicating good generalization  
- Feature selection improved model robustness, while one-hot encoding enhanced predictive performance  

## Skills Demonstrated
- Data preprocessing  
- Feature engineering  
- Regression modeling  
- Model evaluation and validation  
- Handling categorical variables (One-Hot Encoding)  
- Feature selection techniques
