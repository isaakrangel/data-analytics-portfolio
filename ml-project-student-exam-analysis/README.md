# Student Exam Score Analysis & Prediction

## Objective
Analyze and model the factors influencing student exam performance using a structured data science pipeline, including data cleaning, feature engineering, and machine learning modeling.

## Dataset
This dataset contains 20,000 records representing student academic performance, study habits, lifestyle factors, and exam conditions.

Source: https://www.kaggle.com/datasets/kundanbedmutha/exam-score-prediction-dataset

## Project Structure

The project is organized into multiple stages to simulate a real-world data analytics workflow:

### 1. Raw Dataset
- Exam_Score_Prediction.csv  
Original dataset containing raw, unprocessed data.

### 2. Data Cleaning
- Datacleaning.ipynb  
Performed data cleaning, handling missing values, and preparing the dataset for analysis.

### 3. Cleaned Dataset
- Exam_Score_Cleaned_Final.csv  
Final processed dataset used for modeling and analysis.

### 4. Feature Engineering & Modeling
- FeatureEngineerModels.ipynb  
Applied feature engineering, dataset splitting, model training, and evaluation.

## Models Used
- Support Vector Machine (SVM)  
- CatBoost Classifier  
- Other machine learning models depending on experiments  

## Evaluation Metrics
- Accuracy  
- Precision  
- Recall  
- F1-score  

## Process
- Data cleaning and preprocessing  
- Exploratory data understanding  
- Feature selection and transformation  
- Train/test splitting  
- Model training and comparison  
- Performance evaluation  

## Key Insights
- Student performance is influenced by multiple behavioral and lifestyle factors  
- Feature engineering significantly improved model performance  
- Different models showed varying performance depending on feature selection  
- Structured preprocessing pipeline improved reproducibility and clarity
